# Assets Structure (EPANO)

Use `assets/` for static, bundled files only. User content lives in Supabase Storage.

Folders:
- `assets/app/` app icon, splash, adaptive icon, favicon
- `assets/branding/` logo mark, wordmark, OG fallbacks
- `assets/images/empty-states/` empty-state illustrations
- `assets/images/placeholders/` option preview, avatar placeholder, etc.
- `assets/images/backgrounds/` liquid glow shapes, subtle gradients
- `assets/fonts/` local font files (only if we commit to a brand font)
- `assets/lottie/` lottie JSON animations

Keep files optimized, use `kebab-case`, and avoid raw design exports.
