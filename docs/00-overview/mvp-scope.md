# MVP Scope

## In scope
- authentication
- create a trip
- invite members
- add destination or activity suggestions
- vote on suggestions
- create a rough shared expense plan
- create a simple itinerary

## Out of scope
- flight booking
- hotel booking
- in-app payments
- public social feed
- advanced AI trip concierge
- deep native-only features unless necessary

## Success criteria
The MVP is successful if a small friend group can:
1. start planning in one place
2. agree on key decisions faster
3. reduce back-and-forth in chat
4. keep a visible shared trip structure
