# Folder Structure

## app
Contains Expo Router route files and screen composition only.

## src/features
Contains feature-specific logic:
- hooks
- types
- schemas
- api helpers
- stores

## src/components
Contains reusable UI and app components.

## src/lib
Contains shared helpers, constants, utilities, and error helpers.

## src/services
Contains integrations and service wrappers.

## docs
Contains source-of-truth product and architecture documents.

## tests
Contains integration, e2e, and smoke tests.
