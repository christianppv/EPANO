# Core Entities

## User
A person with an account.

## Trip
A planning container for a group trip.

## TripMember
A user linked to a trip, with a role or membership state.

## Suggestion
A destination, activity, or planning proposal added to a trip.

## Vote
A member's input on a suggestion.

## ExpenseItem
A rough shared cost estimate or planned expense.

## ItineraryItem
A dated or ordered plan item for the trip.
